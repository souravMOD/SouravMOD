## Selected Projects

### [GenAI-Pipeline_TrainEfficiency](https://github.com/AI4AgEng-Lab/GenAI-Pipeline_TrainEfficiency)

End-to-end generative pipeline for synthetic training data: **SAM segmentation → Stable Diffusion inpainting → YOLO training**. Reduces manual labeling while maintaining detection accuracy. Accompanies the JSA 2025 paper.

### [Fit_Agent](https://github.com/souravMOD/Fit_Agent)

**Vision-language agent** for fitness and nutrition — analyzes a meal photo and returns calorie estimates with personalized dietary guidance. Demonstrates multimodal perception + reasoning in an agent loop.

### [PDF_RAG](https://github.com/souravMOD/PDF_RAG)

**Retrieval-augmented generation** over PDF documents — ingest a document, ask questions, receive cited answers. Built with LangChain.

### [Object_detection_Huggingface](https://github.com/souravMOD/Object_detection_Huggingface)

Object-detection experiments benchmarking pretrained HuggingFace Transformers on custom datasets.
